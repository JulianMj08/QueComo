# QueComo

Aplicación para gestionar la despensa mediante IA.

## Tecnologías

- FastAPI
- Ollama
- SQLite
- HTML
- CSS
- JavaScript

## Instalación

```bash
git clone https://github.com/JulianMj08/QueComo

cd QComo

python -m venv venv

# Windows
.\venv\Scripts\Activate.ps1

pip install -r requirements.txt

uvicorn app:app --reload
```
